== Description ==

For ProPhoto-theme users, adds international character support in flash slideshow titles

== Installation ==

1. Upload `prophoto-slideshow-internationalization` folder to the `/wp-content/plugins/` directory, or use WordPress auto-plugin-installer.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the ProPhoto customize page and click "save changes" to regenerate static files
